Tyrion
======

[Tyrion](http://www.tyrion.org) is a framework for systems automation.

### Requirements

* [gloox](http://camaya.net/gloox/) >= 1.0
* [SCons](http://www.scons.org/) (build)
* [Twisted](http://twistedmatrix.com/) (test)
